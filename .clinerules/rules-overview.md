# Project Guidelines

## The Rules
- Do not read, write, or modify any files in src/ without express permission.
- Do not modify the contents of docs/inner/
- Do not read, write, or modify any files in bin/
- Do not read, write, or modify any files in obj/
- Do not write to or modify the contents of any folder or subfolder named "old"
- Any file in a folder or subfolder named "old" may only be used for context.
- Never import any Python file in a folder or subfolder named "old".
- All python modules are to be placed in the tools folder.
- Any python script that imports modules in the tools folder is in the root folder.
